The Progress component renders the quality score bar on the design system overview and the confidence bar inside hypothesis-validation HITL cards. It uses the `--color-primary` token for the filled track and `--color-muted` for the background track.

Because `--color-primary` is currently in `drifted` status, any visual rendering of this component reflects the code-side value, not the Figma-side value. Resolving the `color-primary` drift will automatically correct the visual output of this component without a code change.

No Figma mapping has been established. Run `/component Progress` in Claude Code or Cursor to create one.
